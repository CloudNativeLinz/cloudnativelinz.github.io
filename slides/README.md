# README

When adding slides to the `slides/` directory, please follow these guidelines:
1. Add file into the respective sub-directory, e.g. the YEAR-MONTH of the event. Please create it if not yet existing.
2. Name the file in the format in an appropriate way, try to not use white spaces.
3. Update the `/_data/events.yml` file to include the new slides, following the existing format.
4. Submit a pull request for review.
5. Once approved, the slides will be available on the website under the respective event page.

Thank you for contributing!
